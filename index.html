Resumen de tarifas activas

Mercado Pago — Checkout Pro

Para activar el pago con tarjeta necesitás crear un Worker gratuito en Cloudflare que actúa de puente seguro con la API de MP.

⚠️
Worker no configurado
El pago con tarjeta está deshabilitado hasta que configures el Worker.
📋 Pasos para activar
1
Crear cuenta en Cloudflare
Entrá a workers.cloudflare.com → creá una cuenta gratis → "Create a Worker"
2
Copiar el código del Worker
Hacé clic en "Copiar código" y pegalo en el editor de Cloudflare
✓ Copiado
3
Agregar tu Access Token de MP
En Cloudflare → Settings → Variables → New variable → Nombre: MP_ACCESS_TOKEN → Valor: tu Access Token de MP (el que empieza con APP_USR- y es largo)
4
Pegar la URL del Worker acá
Después de publicar el Worker, Cloudflare te da una URL tipo https://mi-worker.workers.dev
